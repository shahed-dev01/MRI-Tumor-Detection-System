# MRI-Tumor-Detection-System

[![Python](https://img.shields.io/badge/Python-3.11-blue?logo=python)](https://www.python.org/) 
[![TensorFlow](https://img.shields.io/badge/TensorFlow-2.x-orange?logo=tensorflow)](https://www.tensorflow.org/) 

**MRI-Tumor-Detection-System** is a deep learning-based project to detect and classify brain tumors from MRI scans. It is designed to help medical professionals in early diagnosis by analyzing MRI images automatically.

---

## Features

- Upload MRI images for analysis
- Automatic brain tumor detection
- Tumor type classification: *Glioma, Meningioma, Pituitary Tumor*
- Simple and interactive interface
- Fast and accurate detection using CNN

---

## Technology Stack

- Python 3.x
- TensorFlow, Keras
- NumPy, OpenCV, Pillow
- Flask (for web interface)
- HTML, CSS, JavaScript (optional frontend)
